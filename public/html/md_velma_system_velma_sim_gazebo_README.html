velma_sim_gazebo

This package defines and implements a group of simulated real effectors of Velma core agent. It is Gazebo plugin and Orocos components for WUT Velma Robot simulation in Gazebo.

It provides ROS launch file:

  • velma_gazebo_re.launch - runs the subsystem and simulator; it uses unpause_on_init.py script to unpause the simulation after initialization procedure is complete
