1 # velma_sim_gazebo {#velma_sim_gazebo_readme}
2 This package defines and implements a group of simulated real effectors of Velma core agent.
3 It is Gazebo plugin and Orocos components for WUT Velma Robot simulation in Gazebo.
4 
5 It provides ROS launch file:
6  * velma_gazebo_re.launch - runs the subsystem and simulator; it uses unpause_on_init script to unpause
7 the simulation after initialization procedure is complete; arguments:
8  * *gui* - run also Gazebo client
9  * *headless* - run Gazebo without graphic card support (all rendering is disabled, even for visual sensors)
10  * *verbose*
11  * *debug* - if `true`, the deployer is executed in gdb.
12  * *world_name* - world file name with respect to GAZEBO_RESOURCE_PATH
13  * *load_robot_description* - if `true`, the launch file loads robot description from urdf and runs robot_state_publisher
14  * *GAZEBO_MODEL_PATH* - please refer to Gazebo documentation
15  * *GAZEBO_RESOURCE_PATH* - please refer to Gazebo documentation
16  * *use_kinect* - `true` or `false`: enable or disable kinect simulation
17  * *ORO_LOGLEVEL* - orocos log level, the higher number the more verbose output
18  * *profile* - physics profile as defined in world file
