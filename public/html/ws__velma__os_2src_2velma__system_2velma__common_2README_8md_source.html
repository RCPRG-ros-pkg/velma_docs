9  * *cam_info_topic*

12  * octomap_offline_server.launch - runs offline octomap server with predefined (saved) map file; arguments:
13  * *octomap_file* - file path for preloaded octomap file. The map sile can be saved using `rosrun octomap_server octomap_saver`
14  * octomap_server.launch - runs the following nodes:
15  * point cloud filter that filters out robot links
16  * online octomap server that uses filtered point cloud to build occupancy map of the environment
17  * octomap filter that adds to the occupancy map unexplored space information
18 
19  arguments:
20  * *use_hw_kinect* - `true` if real HW kinect is used, `false` for simulated kinect
21  * *octomap_file* - file path for preloaded octomap file
